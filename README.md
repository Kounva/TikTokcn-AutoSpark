# 抖音火花助手

## 功能特性

### 账户管理
- 扫码登录（抖音 App 扫码授权）
- 手机号登录
- 手动登录（Base64 Cookie 方式）
- Cookie 一键导出
- 登录状态实时检测
- 上次登录 IP 记录
- 管理员密码修改

### 好友管理
- 好友列表展示（头像、火花天数）
- 好友搜索过滤
- 实时刷新好友数据
- 一键发送消息

### 聊天对话
- 类抖音原生聊天界面（气泡式消息展示）
- 查看好友聊天记录
- 直接发送文字消息
- 支持发送抖音内置表情包（通过 Selenium 自动化操作表情面板）
- 左侧好友列表 + 右侧对话窗口的双栏布局

### 定时任务
- 为好友创建每日定时发送任务
- 支持自定义消息内容（留空使用每日名言）
- 修改已有任务执行时间
- 删除定时任务
- 最近任务快捷入口

### 首页看板
- 浏览器 / 登录状态监测
- 好友数量 / 定时任务数量统计
- 快速操作入口
- 系统运行信息（版本、在线时长、后端地址）

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Vue 3 (Composition API，全局构建，无构建步骤) |
| UI 组件库 | Element Plus + Element Plus Icons |
| 路由 | Vue Router 4 |
| HTTP 客户端 | Axios |
| 后端框架 | FastAPI + Uvicorn |
| 浏览器自动化 | Selenium (Edge WebDriver) |
| 定时调度 | schedule |

**特点**：前端所有依赖（Vue / Element Plus / Axios）已本地化到 `dist/lib/`，无需 npm install，无需构建步骤，启动后端即可使用。

## 快速开始

### 环境要求

- **Python 3.8+**
- **Microsoft Edge 浏览器**（Selenium 自动匹配驱动版本，无需手动下载）
- Windows

### 安装依赖

```bash
pip install -r requirements.txt
```

依赖清单：
- fastapi
- uvicorn
- selenium >= 4.10（内置 Selenium Manager，自动下载驱动）
- schedule
- requests

### 启动服务

```bash
python 抖音自动续火花-后端.py
```

启动后直接输出访问地址，无需任何交互：
- 默认显示浏览器窗口（无头模式易崩溃）
- 端口从 `config.json` 读取（默认 8080），被占用时自动切换到可用端口

启动后访问控制台输出的地址即可使用。

### 使用流程

1. 浏览器打开管理后台 → 用默认账号 `admin / 123456` 登录
2. 首页点击「初始化浏览器」启动 Edge 自动化环境
3. 进入「设置」页 → 扫码或手机号登录抖音
4. 登录成功后回到首页 → 刷新好友列表
5. 在「聊天」页选择好友 → 查看聊天记录、发送消息或表情包
6. 在「定时任务」页为好友添加每日续火花任务

## 端口配置

端口信息保存在 `config.json`：

```json
{
  "port": 8080,
  "show_browser": true
}
```

- 启动时自动读取，被占用时自动切换到可用端口
- 在「设置」页可修改端口和浏览器显示模式，**重启后端后生效**

## 常见问题

### 浏览器崩溃 / 会话断开
- 优先使用**有头模式**（启动时输入任意字符）
- 崩溃后前端会提示「浏览器会话已断开」，点击「初始化浏览器」重启即可

### 登录状态不正确
- 点「设置」页的「刷新状态」按钮，会实际检查抖音页面登录状态
- 如需强制设置登录状态，可使用「强制登录」按钮（调试用）

### 端口被占用
- 启动时自动检测，若端口被占用会自动切换到下一个可用端口
- 控制台会打印实际使用的端口
